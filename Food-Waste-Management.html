Food Waste Management System

A system designed to optimize food usage, reduce waste, and promote sustainability through data tracking and smart resource allocation.

Overview

This project focuses on creating a data-driven platform for tracking and reducing food waste in households and businesses. The system provides insights, recommendations, and visual dashboards to encourage better consumption habits.

Technologies Used

Python, Flask, SQL, JavaScript, and visualization tools for reporting.

Key Outcomes

- Built an interactive dashboard for tracking waste.
- Developed predictive models for food usage.
- Increased awareness of sustainable practices.
